CSS Grid "Masonry"

An attempt at a masonry style layout using CSS Grid and JavaScript.

The 'grid-row-end' property of each grid item is dynamically set until the height of the grid item exceeds that of the content within it.

This is calculated once the page initially loads and recalculated for each grid item once any images within the item have loaded using the imagesLoaded library.

Randomise order of content.

Row height: 20
A larger row height means more white space at the bottom of grid elements but tighter grid packing.

Variety of item width: 1 | 2 | 3
Having all items the same width allows perfect packing.

Grid fill: Dense | Row
Dense packing ensures the least number of gaps but changes the order of the items.
